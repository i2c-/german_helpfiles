###Function###
ACos

###Description###
Errechnet den Arcus-Cosinus einer Zahl.

###Syntax###
ACos ( expression )


###Parameters###
@@ParamTable@@
expression
	Ein Wert zwischen -1 und 1.
@@End@@

###ReturnValue###
Gibt den trigonometrischen Arcus-Cosinus von einer Zahl aus. Das Ergebnis wird als Bogenma&szlig; (Radiant) zur&uuml;ckgegeben (lat. arcus = Bogen)


###Remarks###
<a href="ACos.htm">ACos</a>(x) wird mathematisch nur f&uuml;r -1 <u>&lt;</u> x <u>&lt;</u> 1 definiert, deshalb neigt ACos dazu, -1.#IND f&uuml;r andere Werte von x zur&uuml;ckzugeben.
