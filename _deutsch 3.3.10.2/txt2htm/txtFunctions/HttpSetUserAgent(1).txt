###Function###
HttpSetUserAgent

###Description###
Setzt den Http User-Agent String, der bei allen Internetanfragen mitgesendet wird.

###Syntax###
HttpSetUserAgent ( "user agent" )


###Parameters###
@@ParamTable@@
user agent
	Der String, welcher als User Agent gesetzt wird. &Uuml;bergabe eines leeren Strings wird den User Agent zur&uuml;ck auf Standard (="AutoIt") stellen.
@@End@@

###ReturnValue###
Keine.


###Remarks###
Keine.
