###Function###
IsPtr

###Description###
Testet, ob eine Variable ein Zeigertyp ist.

###Syntax###
IsPtr ( variable )


###Parameters###
@@ParamTable@@
variable
	Die Variable bzw. der Ausdruck, der  &uuml;berpr&uuml;ft werden soll.
@@End@@

###ReturnValue###
@@ReturnTable@@
Erfolg:	1, wenn der Ausdruck vom Typ Zeiger ist.
Fehler:	0, wenn der Ausdruck kein Zeigertyp ist.
@@End@@


###Remarks###
Siehe <a href="../intro/lang_datatypes.htm">Sprachreferenz - Datentypen</a> f&uuml;r eine detaillierte Beschreibung.
