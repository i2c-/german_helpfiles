###User Defined Function###
_GUICtrlRebar_SetUnicodeFormat

###Description###
Setzt das Unicode Zeichen-Format-Flag f&uuml;r das Control

###Syntax###
#include <GuiRebar.au3>
_GUICtrlRebar_SetUnicodeFormat ( $hWnd [, $fUnicode = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle des Rebar-Controls
$fUnicode
	[optional] Entscheidet die zu setzenden Zeichen f&uuml;r das Control:
		True - Das Control verwendet Unicode Zeichen
		False - Das Control verwendet ANSI Zeichen
@@End@@

###ReturnValue###
Keine.
