###Description###

###Parameters###
@@ParamTable@@
$hMenu

$iItem
	ID oder Position des Men&uuml;items
$fState
	[optional] True um den Status zu setzen, sonst False
$fByPos
	[optional] Men&uuml; ID Flag:
		True - $iItem ist eine 0-basierende Itemposition

@@End@@
