###User Defined Function###
_StringRepeat

###Description###
Ein String wird eine bestimmte Anzahl mal wiederholt.

###Syntax###
#include <String.au3>
_StringRepeat ( $sString, $iRepeatCount )


###Parameters###
@@ParamTable@@
$sString
	String, der wiederholt werden soll
$iRepeatCount
	Zahl der Wiederholungen vom String
@@End@@

###ReturnValue###
@@ReturnTable@@
Erfolg:	ein String mit der bestimmten Anzahl
Fehler:	ein leerer String ("") und setzt das @error-Flag ungleich 0
@error:	1 - Einer der Parameter ist ung&uuml;ltig
@@End@@
