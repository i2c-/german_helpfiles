###User Defined Function###
_GUICtrlListBox_SetCaretIndex

###Description###
Setzt das Fokus-Rechteck zum Item an dem festgelegten Index in einer Mehrfachauswahl-Listbox

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_SetCaretIndex ( $hWnd, $iIndex [, $fPartial = False] )


###Parameters###
@@ParamTable@@
$hWnd
	Control-ID / Handle des Controls
$iIndex
	legt den 0-basierenden Index des Items fest
$fPartial
	[optional] Wenn False, wird das Item so lange gescrollt, bis es komplett sichtbar ist.
	Wenn es True ist, bis es zumindest teilweise sichtbar ist
@@End@@

###ReturnValue###
@@ReturnTable@@
Erfolg:	True
Fehler:	False
@@End@@
