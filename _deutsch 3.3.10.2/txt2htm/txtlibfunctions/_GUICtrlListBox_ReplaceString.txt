###User Defined Function###
_GUICtrlListBox_ReplaceString

###Description###
Ersetzt den Text eines Items

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_ReplaceString ( $hWnd, $iIndex, $sText )


###Parameters###
@@ParamTable@@
$hWnd
	Control-ID / Handle des Controls
$iIndex
	legt den 0-basierenden Index des Items fest
$sText
	String, mit dem der bisherige String ersetzt werden soll
@@End@@

###ReturnValue###
@@ReturnTable@@
Erfolg:	True
Fehler:	False
@@End@@
