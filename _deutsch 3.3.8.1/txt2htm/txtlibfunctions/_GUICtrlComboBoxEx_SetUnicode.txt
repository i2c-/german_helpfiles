###User Defined Function###
_GUICtrlComboBoxEx_SetUnicode

###Description###
Legt fest, ob das Control Unicode verwenden soll

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetUnicode($hWnd [, $fUnicode = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle des Controls
$fUnicode
	[optional] kann eines der folgenden sein
		True - schaltet Unicode ein
		False - schaltet  Unicode aus
@@End@@

###ReturnValue###
@@ReturnTable@@
Erfolg:	True
Fehler:	False
@@End@@
