###User Defined Function###
_ClipBoard_GetOpenWindow

###Description###
Gibt das Handle des Fensters zur&uuml;ck, das gerade die Zwischenablage ge&ouml;ffnet hat.

###Syntax###
#include <Clipboard.au3>
_ClipBoard_GetOpenWindow()

###Parameters###
Keine.

###ReturnValue###
@@ReturnTable@@
Erfolg:	Das Handle des Fensters, das gerade die Zwischenablage ge&ouml;ffnet hat
Fehler:	Null, falls kein Fenster die Zwischenablage ge&ouml;ffnet hat
@@End@@

###Remarks###
Falls eine Applikation oder DLL beim Aufruf der Funktion _ClipBoard_Open das Handle eines "NULL-Fensters" spezifiziert, wird zwar die Zwischenablage ge&ouml;ffnet, sie ist aber mit keinem Fenster verbunden. In so einem Fall gibt  _ClipBoard_GetOpenWindow 0 zur&uuml;ck.
