###User Defined Function###
_GUICtrlMonthCal_SetUnicodeFormat

###Description###
Setzt das Unicode Zeichen-Format-Flag f&uuml;r das Control

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_SetUnicodeFormat($hWnd [, $fUnicode = False])

###Parameters###
@@ParamTable@@
$hWnd
	Control-ID / Handle des Controls
$fUnicode
	[optional] Unicode Zeichenflag:

		False - Control verwendet ANSI Zeichen
@@End@@

###ReturnValue###
Erfolg:	Das vorherige Unicode Formatflag
