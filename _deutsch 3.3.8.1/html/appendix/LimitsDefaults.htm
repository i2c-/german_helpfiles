AutoIt3 Standardbegrenzungen


AutoIt3 Begrenzungen Wert Beschreibung
MAX_LINESIZE 4095 Maximale Größe für eine Skriptzeile.
MAX_ENVSIZE 32767 Maximale Größe für eine ENV Variable.
WINTEXTBUFFER 32767 GetWindowText schlägt unter Windows 95 fehl, falls >65535, WM_GETTEXT schlägt manchmal fehl, falls > 32767.
MAXCALLRECURSE 5100 Maximale Anzahl die sich eine Call() Funktion selbst aufrufen kann.
MAXEXECUTERECURSE 5100 Maximale Anzahl die sich eine Execute() Funktion selbst aufrufen kann.
GUI_MAXCONTROLS 65532 Maximale Anzahl von Controls in einem GUI-Fenster.
GUI_MAXPOINTS 256 Maximale Anzahl von Punkten durch eine grafische Infoseite.
COM_MAXEVENT 64 Maximale Anzahl von COM Ereignissen die gepuffert werden können.
VAR_SUBSCRIPT_MAX 64 Maximale Anzahl von Subskripten in einem Array.
MAIN_TIMER_DELAY 750 ms Das verstecken, blinken bzw. zeichnen des Trayicons wird alle 750ms überprüft.
CMDLINEPARAM_MAXLEN 4096 Jeder Parameter kann so viele Zeichen enthalten.
TRAY_MAXITEMS 505 Maximale Anzahl von Items in einem Traymenü.
TRAY_MAXEVENT 32 Maximale Anzahl von Ereignisse die gepuffert werden können.
TRAY_TOOLTIPWIDTH 64 Maximale Anzahl von Zeichen die in einem Tray Tooltip angezeigt werden können.
