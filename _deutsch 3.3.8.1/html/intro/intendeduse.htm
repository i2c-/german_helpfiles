Wozu AutoIt gedacht ist

AutoIt ist eine "wachsende" Skript Sprache. Anfangs wurde es als Add-On Tool verwendet, um grundlegende Aufgaben in GUI`s von anderen Programmen zu übernehmen.
Diese Aufgaben (wie z. B. das Senden von Tasten oder das Klicken auf einen Button) bleiben die Grundlagen-Komponenten eines AutoIt Skripts. Mit der Einbindung der GUI Funktionen kann man nun seine eigene grafische Benutzeroberfläche durch Nutzung von nativen AutoIt Skriptbefehlen erstellen.
Die neue COM (Objekt) Funktionalität füllt die Lücke mit der WSH Sprache wie zum Beispiel VBScript/JScript.

Mit dieser Kombination von GUI Automation, GUI Oberfläche und COM Support stellt AutoIt dem Benutzer ein mächtiges Skript Tool zur Verfügung, das mit vollwertigen Skriptsprachen wie WSH oder KiXStart verglichen werden kann.
